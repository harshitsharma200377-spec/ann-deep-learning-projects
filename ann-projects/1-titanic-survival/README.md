# 🚢 Titanic Survival Prediction — ANN

A deep learning project that predicts whether a passenger **survived or not** on the Titanic using an Artificial Neural Network (ANN).

## 📌 Overview
- **Algorithm:** Artificial Neural Network (ANN)
- **Dataset:** Titanic Dataset (891 samples, 8 features)
- **Task:** Binary Classification (Survived = 1, Did Not Survive = 0)
- **Accuracy:** ~81% on test data

## 🛠️ Tech Stack
- Python, NumPy, Pandas
- TensorFlow, Keras
- Scikit-learn
- Jupyter Notebook

## 📁 Files
- `data/train.csv` — Training dataset
- `data/test.csv` — Test dataset
- `notebooks/Ann_titanic_project.ipynb` — Jupyter notebook with full code
- `requirements.txt` — Python dependencies

## 🚀 Run
```bash
pip install -r requirements.txt
```
Then open `notebooks/Ann_titanic_project.ipynb` in Jupyter or Google Colab and run all cells.
